🍌 首次設定 — 建立資料表

請到你的 Supabase 後台執行以下 SQL，建立遊戲所需的資料表：

-- 貼到 Supabase SQL Editor 執行 create table if not exists rooms ( id uuid default gen_random_uuid() primary key, code text unique not null, host text not null, topic text default '', timer_sec int default 90, phase text default 'lobby', ai_guess text default '', ai_confidence text default '', ai_comment text default '', created_at timestamptz default now() ); create table if not exists players ( id uuid default gen_random_uuid() primary key, room_code text not null, name text not null, avatar_idx int default 0, submitted boolean default false, created_at timestamptz default now(), unique(room_code, name) ); create table if not exists drawings ( id uuid default gen_random_uuid() primary key, room_code text not null, player_name text not null, image_data text not null, created_at timestamptz default now(), unique(room_code, player_name) ); alter table rooms enable row level security; alter table players enable row level security; alter table drawings enable row level security; create policy "allow all" on rooms for all using (true) with check (true); create policy "allow all" on players for all using (true) with check (true); create policy "allow all" on drawings for all using (true) with check (true);
1
前往 Supabase Dashboard，選擇你的專案
2
點左側選單 SQL Editor，貼上以上 SQL，按 Run
3
到 Database → Replication，為 rooms、players、drawings 開啟 Realtime
4
完成後按下方按鈕繼續
